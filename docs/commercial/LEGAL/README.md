# Plantillas legales (borrador)

> **Aviso:** Estos documentos son plantillas para acelerar la entrega al comprador. **Deben ser revisados por asesoría legal cualificada** antes de uso con clínicas en la UE (GDPR) o en República Dominicana.

## Contenido de `LEGAL/`

| Archivo | Propósito |
|---------|-----------|
| `DPA-TEMPLATE.md` | Acuerdo de tratamiento de datos (licenciante ↔ comprador) |
| `PRIVACY-POLICY-TEMPLATE.md` | Política de privacidad para usuarios finales |
| `COOKIE-NOTICE-TEMPLATE.md` | Aviso de cookies (onboarding, sesión, analytics) |
| `HEALTH-DATA-DISCLAIMER.md` | Aviso educativo / no clínico + datos de salud |

## Placeholders a completar

- `[LICENSEE_NAME]` — clínica, asociación o aseguradora compradora
- `[TERRITORY]` — España o República Dominicana
- `[CONTACT_EMAIL]` — DPO o contacto de privacidad
- `[RETENTION_DAYS]` — política de retención de `intake_entries`
- `[EFFECTIVE_DATE]`

## Datos de salud (GDPR)

El diario de ingesta puede constituir **dato de salud** (categoría especial, art. 9 GDPR). El comprador actúa como **responsable del tratamiento** frente a sus usuarios; el licenciante entrega software, no operación SaaS, salvo que se negocie otro modelo.
