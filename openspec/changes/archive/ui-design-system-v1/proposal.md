# Proposal: UI Design System v1

## Intent

Unify repeated layout patterns (headers, stat cards, progress bars, active nav) for visual consistency across Curso, Progreso and Actividad.

## Success Criteria

- [ ] Shared `PageHeader`, `StatCard`, `LevelProgressCard`, `ProgressBar`
- [ ] Nav highlights current route
- [ ] Progress/Analytics/Learn use shared components
- [ ] Mobile-first spacing unchanged or improved
